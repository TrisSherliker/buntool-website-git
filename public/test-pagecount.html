Section 1 — Arbitrary PDFs from directory

Pick a folder of PDFs. Each file is counted by both methods and results compared. These are the two approaches available for reading page counts from existing PDF bytes.

Method A (existing): mupdf.Document.openDocument(bytes, "application/pdf").countPages() — used in makeDummyTocPages.
Method B (pdf-lib): pdflib.PDFDocument.load(bytes).getPageCount() — already in codebase as countPdfPages().

Section 2 — jsPDF-generated documents

Simulates the makeDummyTocPages flow: a jsPDF doc is created, then counted by both methods. The drop-in replacement avoids round-tripping through bytes and mupdf.

Method A (existing): doc.output('arraybuffer') → mupdf.Document.openDocument(…).countPages().
Method B (replacement): doc.internal.getNumberOfPages() — called on the live jsPDF doc before saving.
